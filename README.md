# 盛城房产众筹平台

一个基于 React + Vite + Ant Design 开发的房产众筹平台前端项目。

## 功能特点

- 项目展示：浏览房产众筹项目列表和详情
- 项目管理：管理员可以添加、编辑和删除项目
- 用户中心：查看个人投资记录和账户信息
- 响应式设计：支持多种设备访问

## 技术栈

- React 18
- React Router 6
- Ant Design 5
- Vite 4

## 开发环境设置

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 项目结构

```
src/
  ├── components/     # 公共组件
  ├── data/          # 模拟数据
  ├── pages/         # 页面组件
  └── App.jsx        # 应用入口
```
